# Departemen-Network-UKM-PCC
# Uji Kompetensi Teknik Departemen Network UKM PCC 2026

Repository ini berisi dokumentasi pengerjaan soal **Uji Kompetensi Teknis & Wawancara UKM PCC 2026** pada mata uji:

**Analisis Keamanan Siber (Cyber Security & Network Forensics)**

Ujian ini berbentuk praktik **Capture The Flag (CTF)** dengan fokus pada analisis keamanan jaringan, digital forensics, steganography, dan incident response.

---

## ⚠️ Peringatan Keamanan

Pada soal ini terdapat file investigasi yang berpotensi mengandung **malware/ransomware aktif**, khususnya pada soal Network Forensics.

Oleh karena itu:

- Jangan mengekstrak atau menjalankan file mencurigakan di sistem operasi utama.
- Jangan membuka file executable secara langsung.
- Gunakan lingkungan aman seperti virtual machine atau sandbox.
- Analisis dilakukan secara pasif menggunakan tools forensik, bukan dengan menjalankan malware.

Kesalahan dalam menangani file berbahaya dapat menyebabkan kerusakan pada perangkat pribadi.
